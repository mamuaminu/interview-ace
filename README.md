# Interview Ace

> Security interview prep command center — parse job descriptions, build a question bank, and practice structured answers from your terminal.

## What it does

- **Job Description Parser** — paste a JD, extract relevant security keywords
- **Question Bank** — curated behavioral + technical questions matched to JD keywords
- **Answer Builder** — write and refine your answers
- **STAR Framework Builder** — structure behavioral answers (Situation, Task, Action, Result)
- **Export** — download your prep as a markdown file

## Usage

Open `index.html` in any browser. No server needed.



## Features

- **Keyword extraction** from job descriptions (security-specific dictionary)
- **STAR method** for behavioral questions
- **Question filtering** by type: behavioral, technical, company-specific
- **Local storage** — all data stays in your browser
- **Zero dependencies** — single HTML file, works offline
